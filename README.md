## Installation

```bash
$ npm install
$ npm i @nestjs/config

google login
$ npm i --save @nestjs/passport passport passport-google-oauth2
$ npm install -D @types/passport-google-oauth20

typeorm
$ npm install --save @nestjs/typeorm typeorm pg

mariadb
npm install mysql --save
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev


```
